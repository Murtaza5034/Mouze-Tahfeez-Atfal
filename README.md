# Mauze Tahfeez Atfal App

A modern React application built with Vite for managing student information and daily schedules.

## Features
- **Student Dashboard**: Real-time schedule and school announcements.
- **Profile Management**: Detailed child information card.
- **Responsive Design**: Optimized for mobile and desktop.

## Local Development
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```

## Deployment
This project is configured for one-click deployment to **Vercel**.
The build settings are pre-configured in `vercel.json`.
